Before going live:

1. **Environment Variables** (`.env`):
   - [ ] `DB_PASSWORD` - Strong database password
   - [ ] `GEMINI_API_KEY` - Google AI API key
   - [ ] `JWT_SECRET` - Random 64+ character string
